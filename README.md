# facemood_crawler
-------------------

顏文字卡爬蟲
* 抓取顏文字卡(http://facemood.grtimed.com/index.php) 所有顏文字資料

# 安裝說明
-------------------
* 安裝MySQL資料庫
* 執行`database/facemood.sql`
* 
```
git clone https://github.com/sweslo17/facemood_crawler.git
pip install -r requirements.txt
```
* 修改config.py.template

# 使用說明
-------------------
* `python crawler.py
